❤️ Heart Disease Prediction using Machine Learning


The goal of this project is to build a predictive system that can assist in early detection of heart disease by analyzing patient data. Early prediction can help doctors and healthcare professionals take preventive measures and improve patient outcomes


Heart disease is one of the leading causes of death worldwide. Early detection can help doctors take preventive measures and improve patient health.

In this project, I built a Machine Learning model that predicts whether a person is likely to have heart disease or not based on different medical attributes such as age, cholesterol level, blood pressure, and other health factors.

This project helped me understand how machine learning can be applied in the healthcare domain to assist doctors in making better decisions.

🎯 Objective

The main objective of this project is to develop a predictive model that can analyze patient health data and determine the risk of heart disease.

📊 Dataset

The dataset contains several medical attributes used for prediction, such as:

Age

Sex

Chest Pain Type

Resting Blood Pressure

Cholesterol Level

Maximum Heart Rate

Fasting Blood Sugar

Exercise Induced Angina

ST Depression

Target (Heart Disease or No Heart Disease)
